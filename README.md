# ASCII QUERY

## Compile

```bash
gcc -o asc_query asc_query.c
```

## Usage

```
Usage: ascquery [options] character.

This program can show the ascii code of the character, also,
 it can show the char of the character.

OPTIONS:
    -i, --int: get ascii code of the character.
    -c, --char: get char of the given ascii code.
    -x, --hex: the given ascii code is hex, or output the ascii code as hex.
    -h, --help: show this message.
```
